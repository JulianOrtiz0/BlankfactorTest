# BlankfactorTest

##  Setup & Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/JulianOrtiz0/BlankfactorTest.git
    cd blankfactor-automation
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

##  Running the Tests

You can run the tests using either the graphical interface or the command line:

### Interactive Mode (UI)

```bash
npx cypress open
```

### CLI

```bash
npx cypress run
```
